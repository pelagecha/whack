    ./setup.sh
    ```
    or
    ```
    ./setup.ps1
    ``` depending on your os.

## After Setup

Run in two different terminals ``` 
                                ./run_flask(.sh/.ps1)
                                ```
                                and
                                ``` 
                                ./run_next(.sh/.ps1)
                                ```.
