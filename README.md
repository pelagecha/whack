# whack



## Setup using bash
Run ```
    ./setup.sh
    ```
    for MacOs and Linux, or
    ```
    ./setup.ps1
    ```
    for Windows.


## After Setup

Run in two different terminals 
    ``` 
    ./run_flask(.sh/.ps1)
    ```
    and
    ``` 
    ./run_next(.sh/.ps1)
    ```

### Future testing

-   **Backend:**
    ```
    -   cd backend/
    -   python app.py
    ```

-   **Frontend:**
    ```
    -   cd frontend/
    -   npm run dev
    ```
