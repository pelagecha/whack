# whack

## Setup
Run ```bash
    ./setup.sh
    ```
    or
    ```bash
    ./setup.ps1
    ``` depending on your os.

## After Setup

Run in two different terminals ```bash 
                                ./run_flask(.sh / .ps1)
                                ```
                                and
                                ```bash 
                                ./run_next(.sh / .ps1)
                                ```.
